fooTransition( className, state, callback [, timeout ] ) → {jQuery}

Add or remove one or more class names that trigger a transition and perform an action once it ends.

Description

This exposes the FooGallery.utils.transition.start method as a jQuery plugin.

Parameters
Name Type Attributes Description
className string

One or more class names (separated by spaces) to be toggled that starts the transition.
state boolean

A Boolean (not just truthy/falsy) value to determine whether the class should be added or removed.
callback function

A function to call once the transition is complete.
timeout number <optional>

The maximum time, in milliseconds, to wait for the transitionend event to be raised. If not provided this will be automatically set to the elements transition-duration property plus an extra 50 milliseconds.
Returns
Examples
jQuery( ".selector" ).fooTransition( "expand", true, function(){
	// do something once the transition is complete
} );
